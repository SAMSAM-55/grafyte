# Changelog

## 0.3.3

### New features

- Added the ability to create multiple scenes and UIs in a single application.

### Improvements

- Improved the GitHub Actions CLI workflows, including PR matrix fail-fast behavior.
- Added and fixed CLI test coverage for pull requests.
- Updated package metadata and tightened the supported Python version range.
- Cleaned up typos and stubs in docs and type hints.

## 0.3.2

### Improvements

- Refined the text and camera APIs to use read/write properties.
- Updated the README, contributing guide, and rendering internals.
- Added unit tests for the Python wrapper layer.

### Fixes

- Fixed camera culling and object scaling behavior in the renderer.

## 0.3.1

### Fixes

- Fixed a rendering bug caused by null scene or UI references.
